## Google Search Console Setup Steps

### 1. Verify Domain Ownership

**Option A: HTML Meta Tag**
Add this to your `app/layout.tsx` (already configured):
```html
<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" />
```

**Option B: DNS Record**
Add a TXT record to your domain DNS settings

**Option C: HTML File Upload**
Upload the verification file to your site root

### 2. Submit Sitemap

1. Go to Google Search Console
2. Navigate to **Sitemaps** section
3. Enter: `https://www.homesteadwestlasvegas.com/sitemap.xml`
4. Click **Submit**
