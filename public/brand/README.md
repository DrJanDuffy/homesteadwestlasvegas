## Obtaining the official BHHS logo

1. Contact your Berkshire Hathaway HomeServices Nevada broker.
2. Request logo files from Global Branding: GlobalBranding@HSFranchise.com
3. Use brand resources from the BHHS agent portal where applicable.

**File requirements:** PNG or SVG, ~300px width minimum, transparent background preferred.
